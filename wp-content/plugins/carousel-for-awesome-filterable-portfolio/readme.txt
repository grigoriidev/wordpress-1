== Description ==

!!! You can also contact us or email us for further customization !!!
*Please submit a review if you like the plugin*

This is a wrapper plugin for the elastislide http://tympanus.net/codrops/2011/09/12/elastislide-responsive-carousel/ script (with some modifications)

= Features =

* This plugin works in conjunction with Awesome Filterable Portfolio plugin.
* Automatically shows all the thumbnail portfolio images into your elastic slider carousel.
* Fully responsive.
* Shortcode options available.
* Resources are loaded ONLY if slider is present on the page.

= Use shortcode in PHP file = 

<?php echo do_shortcode('[af_carousel]');?>

Original Script: http://tympanus.net/codrops/2011/09/12/elastislide-responsive-carousel/

== Installation ==

1. Download plugin and upload contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert the = [af_carousel] = shortcode where you want the carousel to appear.
